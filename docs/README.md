- 새 기능이 생기면 `docs/user_guide.md`를 먼저 수정합니다.
- 새 버전을 배포할 때는 PDF를 다시 생성하고 파일명과 링크의 버전 번호를 갱신합니다.
- v2.0.1 기능 설명은 최신 Markdown 사용자 가이드를 기준으로 합니다.
